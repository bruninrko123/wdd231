AI compass Website Plan

Site name: AI compass

Site Purpose: Teaching people how to use different AI tools effectively and presenting AI tools that have varied utilities.

Scenarios:

  • What AI tools can I use to help me write, create presentations, write and review code, practice language skills and so on?
  • What are the main features of each of those apps?
  • Are they free? if not, where can I find the price?
